算法：一个环上有 10 个点,编号为 0-9,从 0 点出发,每步可以顺时针到下一个点,也可以逆时针到上一个点,求:经过 n 步又回到 0 点有多少种不同的走法



举例：



如果 n=1，则从 0 出发只能到 1 或者 9，不可能回到 0，共 0 种走法



如果 n=2，则从 0 出发有 4 条路径:0->1->2, 0->1->0, 0->9->8, 0->9->0,其中有两条回到了 0 点，故一共有 2 种走法
